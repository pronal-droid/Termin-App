Google Cloud Einrichtung:
1. Geh zu console.cloud.google.com
2. Erstelle ein neues Projekt (z.B. "Termin-App")
3. Geh zu "APIs & Services" → "Library" → suche "Google Calendar API" → Aktivieren
4. Geh zu "APIs & Services" → "Credentials" → "Create Credentials" → "OAuth client ID"
5. Application type: "Web application"
6. Bei "Authorized JavaScript origins" deine GitHub Pages URL eintragen (z.B. https://DEIN-NAME.github.io)
7. Bei "Authorized redirect URIs" auch deine GitHub Pages URL eintragen
8. Client-ID kopieren und oben einfügen
9. Geh zu "OAuth consent screen" → "Test users" → füge deine eigene Gmail-Adresse hinzu

Wichtig: Solange die App im "Testing"-Modus ist, können sich nur die eingetragenen Test-User anmelden.
